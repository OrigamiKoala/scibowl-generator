# Science Bowl Packet Writer

This is a skill with instructions for an AI agent to write Science Bowl packets. It is intended for use with Google's Antigravity agent. However, with minimal tweaking, the skill can easily be imported to work with any other AI agent.

Credits:

The reference files sent to the agent contain TeX versions of the MIT 2025 Science Bowl Packets. Find out more about MIT Science Bowl on their [website](https://www.mitsciencebowl.com/home).
